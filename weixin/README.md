# 微信小程序 - Java1234个人健康管理系统

## 使用说明

1. 微信开发者工具导入本项目（`weixin` 目录）
2. 详情 → 本地设置 → 勾选「不校验合法域名、web-view、TLS 版本」
3. 确保后端服务已启动（http://localhost:8080）

## 页面

| 页面 | 功能 |
|------|------|
| 登录 | 用户名密码登录/注册 |
| 首页 | 今日健康概览、快捷入口 |
| 健康 | 档案/饮食/运动/睡眠记录 |
| 资讯 | 健康科普文章 |
| 我的 | 个人信息、退出登录 |
| AI顾问 | 智能健康客服（百炼大模型） |
| 提醒 | 健康提醒管理 |

## 后端地址配置

修改 `miniprogram/utils/request.ts` 中的 `BASE_URL`：

```typescript
const BASE_URL = 'http://localhost:8080/api'
```

真机调试时需改为局域网 IP 地址。
